## 一、排名系统概览

Google 的排名系统使用**数百个因素和信号**，主要系统包括：

### 核心排名系统

- 在网页级别运行，使用各种信号了解如何排名
- 也使用**网站级信号**辅助
- 良好的网站级信号 ≠ 所有内容都高排名
- 许多曾经独立的系统（熊猫、企鹅、有用内容）已并入核心系统

### 重要子系统

| 系统 | 作用 |
|------|------|
| **有用内容系统** | 确保用户看到面向用户的原创内容（2024年3月已并入核心系统） |
| **链接分析 + PageRank** | 了解网页间链接关系，判断内容实用性 |
| **BERT** | 了解字词不同组合如何表达不同含义 |
| **RankBrain** | 了解字词与概念关系，即使不含确切搜索词也能返回相关内容 |
| **神经匹配** | 理解查询和网页中概念的表示形式并匹配 |
| **段落排名** | AI 识别网页各段落，更好了解相关性 |
| **原创内容系统** | 确保原创报道排在引用内容前面 |
| **更新系统** | 确保时效性内容优先展示 |
| **可靠信息系统** | 提升权威网页排名，降低劣质内容排名 |
| **评价系统** | 奖励包含见解深刻的分析和原创研究的优质评价 |
| **网站多元化系统** | 一般不显示同一网站2个以上结果 |

## 二、核心更新 (Core Updates)

### 什么是核心更新

- Google 每年进行多次大规模搜索算法变更
- **不是针对特定网站**，是宽泛变更
- 类比：就像餐厅排行榜，新餐厅出现后旧排名下滑，不是旧餐厅变差了

### 应对排名下降

1. **确认更新状态** — 查看搜索状态信息中心
2. **等待一周再比较** — 更新完成后至少等一周再分析数据
3. **评估下降幅度**：
   - 小幅下降（2→4位）：不需重大措施
   - 大幅下降（4→29位）：需深入评估
4. **细分搜索类型** — 单独分析网页/图片/视频/新闻
5. **全面客观评估** — 让第三方人员评估

### 内容改进建议

| 要点 | 说明 |
|------|------|
| 拒绝"快速修复" | 不要盲目移除某些元素，专注于长期有效改进 |
| 有意义地改进 | 重写或重组内容结构，让读者更轻松 |
| 谨慎删除内容 | 仅在内容完全无法挽回时考虑 |
| 耐心等待 | 改进后需几天到几个月才见效 |
| 不必死等大更新 | Google 也持续进行较小规模更新 |

## 三、技术 SEO 基本要求

### 确保 Google 能抓取

| 检查项 | 说明 |
|--------|------|
| 服务器稳定 | 确保正常处理请求 |
| 网络连通 | 无 DNS 或防火墙问题 |
| robots.txt | 未意外屏蔽 Googlebot |
| 无需登录 | Googlebot 无法登录 |

### 确保 Google 能索引

| 检查项 | 说明 |
|--------|------|
| 内容质量 | 质量低的内容不会被索引 |
| noindex 标记 | 确认未误用 |
| 网站设计 | 确保易于被索引 |

### 确保 Google 能呈现

| 检查项 | 说明 |
|--------|------|
| 内容相关性 | 内容是否与目标查询相关 |
| 质量达标 | 内容质量是否达到展示标准 |
| meta 规则 | 未被阻止呈现 |

## 四、JavaScript SEO

### Google 如何处理 JS

- 使用**最新版 Chrome** 渲染网页并运行 JavaScript
- 这意味着 Google 能看到 JS 生成的内容
- 但渲染可能有**延迟**（先抓取 HTML，后续再渲染 JS）

### 最佳实践

1. **关键内容放在 HTML 中** — 不要完全依赖 JS 渲染
2. **使用 SSR（服务器端渲染）** — 对 SEO 最友好
3. **确保懒加载内容可发现** — 使用正确的懒加载技术

### 核心功能

| 功能 | 用途 |
|------|------|
| 网址检查工具 | 查看 Google 看到的页面样貌 |
| 效果报告 | 监控搜索流量和排名变化 |

### 调试流量下降

1. 检查是否有核心更新
2. 在 Search Console 中按日期对比数据
3. 细分不同搜索类型分析
4. 客观评估内容质量

## 六、网络垃圾政策（红线）

### 绝对禁止

- **关键字堆砌** — 在内容中不自然地重复关键词
- **隐藏文字/链接** — 用户看不到但搜索引擎能看到
- **门页 (Doorway Pages)** — 为特定搜索词创建大量相似页面
- **抄袭内容** — 复制他人内容无额外价值
- **伪装 (Cloaking)** — 向搜索引擎和用户展示不同内容
- **付费链接** — 购买链接提升排名
- **自动生成垃圾内容** — AI 生成的内容无用户价值

### 违规后果

- Google 可能对网站执行**人工处置措施**
- 排名大幅下降或从索引中移除
- 解决问题后可提交重新审核
