# 删除文章

## 接口信息
- **URL**: `/api/article/delete/`
- **Method**: `POST`
- **Content-Type**: `application/x-www-form-urlencoded`

## 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| id | string | 是 | - | 文章 ID |

## 响应示例

### 成功 (code=0)
```json
{
    "code": 0,
    "message": "文章删除成功",
    "data": {}
}
```

### 失败 - 文章不存在 (code=2001)
```json
{
    "code": 2001,
    "message": "文章不存在",
    "data": None
}
```

## 状态码说明

| code | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 必填参数缺失 |
| 1002 | 参数不合法（ID非数字） |
| 2001 | 文章不存在 |
| 2005 | 文章删除失败（数据库异常） |
| 2008 | 文章文件删除失败 |

## 备注
- 物理删除：同时删除磁盘上的 HTML 文件和数据库记录
- 删除操作不可逆，请谨慎调用
