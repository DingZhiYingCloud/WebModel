# 站点地图生成器使用教程

> 模块路径：`utils/模块文件数据库/sitemap_generator.py`

## 概述

标准 XML sitemap 生成工具，输出符合 [sitemaps.org 协议](https://www.sitemaps.org/protocol.html) 的 XML 文件，供 Google / Bing 等搜索引擎抓取。

**特点**：
- 纯 Python 实现，不依赖 Django（数据通过参数传入，方便复用和测试）
- 支持 `<loc>`、`<lastmod>`、`<changefreq>`、`<priority>` 四个标准字段
- 提供 CLI 命令行接口

## 环境变量

站点地图需要**公开可访问的完整 URL**，由 `settings.py` 从 `.env` 的 `ALLOWED_HOSTS` 自动推导，无需单独配置：

```env
# 允许的主机（第一个条目用于构建 SITE_URL）
# 本地开发：127.0.0.1 → 自动推导为 http://127.0.0.1:8000
# 生产环境：example.com → 自动推导为 https://example.com
ALLOWED_HOSTS=127.0.0.1

# 调试日志（True 时输出 loguru 日志）
DEBUG=True
```

**推导规则**（在 `WebModel/settings.py` 中自动执行）：
- `ALLOWED_HOSTS=127.0.0.1` / `localhost` / `0.0.0.0` → `http://{host}:8000`
- `ALLOWED_HOSTS=example.com` → `https://example.com`

## 快速开始

### 1. Python 调用

```python
import sys
sys.path.insert(0, r"A:\MyProjects\WebModel\utils\模块文件数据库")
from sitemap_generator import SitemapGenerator

gen = SitemapGenerator(base_url="https://example.com")

articles = [
    {"loc": "/web/youdaotools/news/1/", "lastmod": "2026-04-25"},
    {"loc": "/web/youdaotools/news/2/", "lastmod": "2026-04-26"},
    # 也可覆盖单条的频率和优先级
    {"loc": "/web/youdaotools/news/3/", "lastmod": "2026-04-27", "changefreq": "daily", "priority": "1.0"},
]

xml_string = gen.generate(articles)
print(xml_string)
```

### 2. Django 视图中使用

项目已内置 sitemap 视图，直接访问即可：

```
GET /web/{SITE_SLUG}/sitemap.xml
```

例如当前配置下：`http://127.0.0.1:8000/web/youdaotools/sitemap.xml`

### 3. 命令行使用

```bash
# 切到模块目录
cd A:\MyProjects\WebModel\utils\模块文件数据库

# 传入 JSON 字符串
python sitemap_generator.py --base-url https://example.com --articles '[{"loc":"/news/1/","lastmod":"2026-04-25"}]'

# 从 JSON 文件读取
python sitemap_generator.py --base-url https://example.com --articles articles.json --output sitemap.xml

# 自定义频率和优先级
python sitemap_generator.py --base-url https://example.com --articles articles.json --changefreq daily --priority 0.9
```

## 核心 API

### `SitemapGenerator`

| 方法 | 说明 | 参数 | 返回值 |
|------|------|------|--------|
| `__init__(base_url, default_changefreq, default_priority)` | 初始化生成器 | base_url: 站点根URL; default_changefreq: 默认频率(默认weekly); default_priority: 默认优先级(默认0.8) | - |
| `generate(articles)` | 生成 sitemap XML | articles: `[{"loc","lastmod","changefreq","priority"}]` | XML 字符串 |

### articles 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `loc` | 是 | 页面路径（相对路径，会与 base_url 拼接） | `/web/youdaotools/news/1/` |
| `lastmod` | 否 | 最后修改时间 | `2026-04-25` |
| `changefreq` | 否 | 更新频率，覆盖默认值 | `daily` |
| `priority` | 否 | 优先级 0.0~1.0，覆盖默认值 | `0.9` |

### changefreq 可选值

| 值 | 含义 |
|----|------|
| `always` | 每次访问都更新 |
| `hourly` | 每小时 |
| `daily` | 每天 |
| `weekly` | 每周 |
| `monthly` | 每月 |
| `yearly` | 每年 |
| `never` | 从不更新（归档内容） |

## 输出示例

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/web/youdaotools/news/1/</loc>
    <lastmod>2026-04-25</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://example.com/web/youdaotools/news/2/</loc>
    <lastmod>2026-04-26</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>
```

## 错误处理

- **base_url 为空**：初始化时抛出 `ValueError`
- **文章列表为空**：`generate()` 抛出 `ValueError`
- **某条记录缺少 loc**：跳过该条，输出 warning 日志

## 依赖

- `loguru` - 日志（可选，仅 DEBUG=True 时使用）
