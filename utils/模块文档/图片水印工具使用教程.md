# 图片水印工具使用教程

> 模块路径：`utils/模块文件数据库/image_watermark.py`

## 概述

通用的图片下载 + 动态文字水印工具，支持单张和批量处理。水印文本和位置从项目根目录 `.env` 中读取，也可运行时覆盖。

## 环境变量

在项目根目录 `.env` 中配置：

```env
# 水印文字
ARTICLE_IMAGE_WATERMARK_TEXT=有道翻译
# 水印位置：左上角 | 右上角 | 左下角 | 右下角 | 居中
ARTICLE_IMAGE_WATERMARK_POSITION=右下角
# 调试日志（True 时输出 loguru 日志）
DEBUG=True
```

## 快速开始

### 1. Python 调用

```python
import sys
sys.path.insert(0, r"A:\MyProjects\WebModel\utils\模块文件数据库")
from image_watermark import WatermarkProcessor

# 使用 .env 默认配置
wp = WatermarkProcessor()

# 单张图片：下载 + 水印
result = wp.download_and_watermark(
    image_url="https://example.com/photo.jpg",
    output_dir="./media/articles/my-article",
    filename="photo.jpg",
)
print(result.success, result.local_path)

# 批量处理
images = [
    {"url": "https://example.com/a.jpg", "filename": "a.jpg"},
    {"url": "https://example.com/b.png", "filename": "b.png"},
]
results = wp.batch_process(images, "./media/articles/my-article")
for r in results:
    print(r.filename, r.success, r.local_path)
```

### 2. 自定义水印配置

```python
from image_watermark import WatermarkProcessor, WatermarkConfig

# 方式一：传入参数覆盖
wp = WatermarkProcessor(text="版权所有", position="居中")

# 方式二：使用完整的 WatermarkConfig
config = WatermarkConfig(
    text="内部资料",
    position="左下角",
    opacity=0.8,           # 透明度 0.0~1.0
    font_size_ratio=0.05,  # 字体大小占短边比例
    padding_ratio=0.04,    # 边距占短边比例
)
wp = WatermarkProcessor(config=config)
```

### 3. 为本地已有图片添加水印

```python
wp = WatermarkProcessor()
wp.add_watermark("./original.jpg", "./watermarked.jpg")
```

### 4. 命令行使用

```bash
# 切到模块目录
cd A:\MyProjects\WebModel\utils\模块文件数据库

# 单张图片
python image_watermark.py --url https://example.com/img.jpg --output-dir ./out --filename photo.jpg

# 批量处理
python image_watermark.py --images '[{"url":"https://a.com/1.jpg","filename":"1.jpg"},{"url":"https://b.com/2.png","filename":"2.png"}]' --output-dir ./out

# 自定义水印
python image_watermark.py --url https://example.com/img.jpg --output-dir ./out --filename photo.jpg --text "自定义水印" --position 居中
```

## 核心 API

### `WatermarkProcessor`

| 方法 | 说明 | 参数 | 返回值 |
|------|------|------|--------|
| `__init__(text, position, config)` | 初始化处理器 | text: 水印文字; position: 位置; config: 完整配置 | - |
| `download_and_watermark(url, output_dir, filename)` | 下载图片+水印+保存 | url: 图片地址; output_dir: 保存目录; filename: 文件名 | `WatermarkResult` |
| `batch_process(images, output_dir)` | 批量处理 | images: `[{"url","filename"}]`; output_dir: 保存目录 | `List[WatermarkResult]` |
| `add_watermark(image_path, output_path)` | 本地图片加水印 | image_path: 原图路径; output_path: 输出路径 | 保存路径字符串 |

### `WatermarkResult`

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | bool | 是否成功 |
| `local_path` | str | 本地保存路径 |
| `filename` | str | 文件名 |
| `original_url` | str | 原始 URL |
| `error` | str | 错误信息（失败时） |

### `WatermarkConfig`

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `text` | .env 配置 | 水印文字 |
| `position` | .env 配置 | 水印位置 |
| `font_size_ratio` | 0.04 | 字体大小占图片短边的比例 |
| `opacity` | 0.6 | 水印透明度 (0.0~1.0) |
| `color` | (255,255,255) | 水印颜色 RGB |
| `stroke_width` | 1 | 描边宽度 |
| `stroke_color` | (0,0,0) | 描边颜色 RGB |
| `padding_ratio` | 0.03 | 边距占短边比例 |

## 支持的水印位置

| 中文 | 英文 | 效果 |
|------|------|------|
| 左上角 | top-left | 左上角 |
| 右上角 | top-right | 右上角 |
| 左下角 | bottom-left | 左下角 |
| 右下角 | bottom-right | 右下角 |
| 居中 | center | 正中央 |

中文和英文位置名均可使用。

## 错误处理

- **图片下载失败**：`download_and_watermark` 返回 `success=False`，包含 `error` 描述
- **批量处理中断**：`batch_process` 中任一图片失败即抛出 `ValueError`，包含已成功数量
- **字体缺失**：自动降级到 PIL 默认字体（中文可能显示为方框），并输出警告日志
- **水印文字为空**：初始化时直接抛出 `ValueError`

## 依赖

- `Pillow` (PIL) - 图片处理
- `requests` - 图片下载
- `loguru` - 日志（可选，仅 DEBUG=True 时使用）
